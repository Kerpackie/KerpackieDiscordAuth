# Kerpackie Discord Auth

Welcome to the **Kerpackie Discord Auth** documentation.

This library provides Discord authentication integration for ASP.NET Core applications, including support for ASP.NET Core Identity.

## Features

- **Discord Authentication**: Easy integration with Discord's OAuth2 flow.
- **Identity Support**: Seamless integration with ASP.NET Core Identity (`Kerpackie.Discord.Auth.Identity`).
- **Customizable**: Flexible configuration options for scopes, claims, and more.
